# Hakner Raytracer
This is a project for me to learn the ins and outs of modern raytracing on both CPU and later GPU.

### Current Features
* Sphere Intersection
* BVH

### Dependencies
* SDL2
* DirectXMath ( + SimpleMath wrapper)
* Dear ImGUI
* STB_Image, STB_Image_Write
